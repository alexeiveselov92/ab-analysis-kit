abkit brand system · v0.1
Identity & design direction

Call it once it stabilizes.

abkit is the sibling of detectkit — same warm, engineering-calm DNA, its own hue. It turns an A/B experiment into a cumulative effect + confidence interval you can watch converge, then read off a clean verdict. The system has to feel just as sharp in a chat readout as on a docs page.
